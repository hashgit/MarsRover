The solution consists of two projects.

- RoverMover

This project contains all the code and classes to execute a command on Rover.
Following classes are part of the project:

* Plateau
* Rover
* Command
* MoveCommand
* LeftCommand
* RightCommand
* CommandFactory

- RoverMover.Tests

This project contains some test cases to check the validity of the program.
Run the tests ScenarioOne and ScenarioTwo to get the desired results.

The project contains only sample test cases, a lot more test cases can be written for this.
